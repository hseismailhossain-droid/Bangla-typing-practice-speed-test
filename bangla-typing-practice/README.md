# 🇧🇩 Bangla Typing Practice (বাংলা টাইপিং প্র্যাকটিস)

একটি আধুনিক এবং অত্যন্ত দৃষ্টিনন্দন বাংলা টাইপিং শেখার ও প্র্যাকটিস করার ওয়েব অ্যাপ্লিকেশন। এটি ব্যবহার করে যে কেউ সহজে **অভ্র ফনেটিক (Avro Phonetic)** এবং **বিজয় (Bijoy)** কিবোর্ড লেআউট অনুশীলন করতে পারবেন। অ্যাপ্লিকেশনটিতে রয়েছে লেসন বা অধ্যায়ভিত্তিক অনুশীলন ব্যবস্থা, রিয়েল-টাইম স্পিড (WPM), গতি ও ভুলের হিসাব, মেডেল অর্জন এবং ভেরিফাইড সার্টিফিকেট ডাউনলোড ব্যবস্থা।

---

## 🚀 মূল বৈশিষ্ঠ্যসমূহ (Features)

*   **মাল্টি-কিবোর্ড লেআউট:** অভ্র (Avro Phonetic) এবং বিজয় (Bijoy) লেআউট সমর্থন।
*   **সহায়ক গাইড (Interactive Hints):** ইংরেজি কিবোর্ডের কোন বাটন টিপলে কোন বাংলা অক্ষরটি আসবে, তার বড় ও স্পষ্ট ভিজ্যুয়াল গাইডলাইন।
*   **মজাদার অনুশীলন গেম:** বিভিন্ন স্তরের টাইপিং চ্যালেন্জিং গেম।
*   **সনদপত্র জেনারেটর (Official Certificate):** নাম লিখে ৬০ সেকেন্ডের একটি পরীক্ষা দিয়ে সুন্দর পিডিএফ প্রিন্ট বা সরাসরি লোকাল ডিস্কে **উচ্চমানের PNG ইমেজ** হিসেবে সার্টিফিকেট ডাউনলোডের সুবিধা।
*   **কনফেত্তি অ্যানিমেশন:** নতুন সার্টিফিকেট অর্জন বা ডাউনলোড করার সময় চমৎকার ফায়ারওয়ার্কস কনফেত্তি এফেক্ট।
*   **সম্পূর্ণ অফলাইন অফলাইন-ফার্স্ট স্পিড ট্র্যাকিং:** কোনো সার্ভার বা ইন্টারনেটের ঝামেলা ছাড়াই লোকাল স্টোরেজে ডেটা সংরক্ষণ।

---

## 🛠️ লোকাল রান করার নিয়ম (Local Installation & Running)

১. প্রথমে আপনার কম্পিউটারে **Node.js** ইন্সটল করা না থাকলে তা ইন্সটল করে নিন।
২. আপনার টার্মিনালে নিচের কমান্ড রান করে প্রোজেক্টের ডিপেন্ডেন্সি ডাউনলোড করুন:
   ```bash
   npm install
   ```
৩. এরপর লোকাল ডেভেলপমেন্ট সার্ভার চালু করতে রান করুন:
   ```bash
   npm run dev
   ```
৪. এটি লোকালহোস্টে `http://localhost:3000` পোর্টে অ্যাপ্লিকেশনটি চালু করবে।

---

## 📂 প্রোজেক্টের ডিরেক্টরি কাঠামো (Directory Structure)

*   `/src/App.tsx` - অ্যাপটির মূল লেআউট এবং রাউটিং কাঠামো।
*   `/src/components/` - মডুলার রিয়্যাক্ট কম্পোনেন্টস (যেমন: `CertificateGenerator`, `TypingGameModule`, `AlphabetSelector` ইত্যাদি)।
*   `/src/utils/` - অভ্র ও বিজয় টাইপিং রূপান্তর লজিক এবং সাউন্ড ক্লিপ হ্যান্ডলার।
*   `/vercel.json` - ভার্সেল (Vercel) ক্লাউড হোস্টিং কনফিগারেশন ফাইল।

## 🐙 ১. GitHub-এ আপলোড করার নিয়ম

প্রোজেক্টটি গিটহাবে পুশ করার জন্য নিচের ধাপগুলো অনুসরণ করুন:

```bash

git remote add origin https://github.com/YOUR_USERNAME/bangla-typing-practice.git

# গিটহাবে পুশ করুন
git push -u origin main
```

## ⚡ ২. Vercel-এ হোস্টিং করার নিয়ম (Vite Vercel Deployment)

এই প্রজেক্টটি সরাসরি **Vercel**-এ বিনামূল্যে হোস্ট করা খুবই সহজ:

১. [Vercel](https://vercel.com/) এ লগইন বা সাইন-আপ করুন।
২. **Add New Project** বোতামে ক্লিক করুন।
৩. আপনার লিঙ্কড **GitHub** অ্যাকাউন্ট থেকে `bangla-typing-practice` রিপোজিটরি সিলেক্ট করুন।
৪. Vercel স্বয়ংক্রিয়ভাবে এটি একটি **Vite project** হিসেবে ডিটেক্ট করবে।
৫. **Build and Output Settings** পরিবর্তন করার প্রয়োজন নেই (এটি `npm run build` এবং `dist` ডিরেক্টরি ব্যবহার করবে)।
৬. **Deploy** বোতামে ক্লিক করুন। এক মিনিটের মধ্যে আপনার সাইট লাইভ হয়ে যাবে!

## 📱 ৩. Google Play Store-এ পাবলিশ করার নিয়ম (Android App Packing)

এই চমৎকার রিয়্যাক্ট-ভ্যাইট অ্যাপটিকে আপনি সহজেই **Capacitor** ব্যবহার করে একটি নেটিভ অ্যান্ড্রয়েড অ্যাপে রূপান্তর করতে পারেন এবং গুগল প্লে স্টোরে রিলিজ করতে পারেন।

### ধাপ-১: প্রথমত Capacitor প্রোজেক্টে ইনস্টল করুন:
```bash
npm install @capacitor/core @capacitor/cli
```

### ধাপ-২: Capacitor কনফিগার করুন:
```bash
npx cap init
```
*   **App Name:** Bangla Typing Practice
*   **Package ID:** com.yourname.banglatyping (যেমন: `com.barakaship.banglatyping`)
*   **Web Asset Directory:** `dist` (এটি অত্যন্ত গুরুত্বপূর্ণ, কনফিগারেশনে `dist` লিখুন)

### ধাপ-৩: অ্যান্ড্রোয়েড প্ল্যাটফর্ম যুক্ত করুন:
```bash
# অ্যান্ড্রোয়েড প্যাকেজ ইনস্টল করুন
npm install @capacitor/android

# অ্যান্ড্রোয়েড প্ল্যাটফর্ম ক্রিয়েট করুন
npx cap add android
```

### ধাপ-৪: আপনার কোডের লেটেস্ট কন্টেন্ট বিল্ড এবং সিঙ্ক করুন:
```bash
# প্রজেক্ট বিল্ড করুন (dist ফোল্ডার জেনারেট হবে)
npm run build

# বিল্ট রিসোর্স অ্যান্ড্রোয়েডে কপি করুন
npx cap sync
```

### ধাপ-৫: Android Studio দিয়ে অ্যাপ রান অথবা রিলিজ করুন:
```bash
npx cap open android
```
এটি **Android Studio**-তে আপনার অ্যান্ড্রোয়েড প্রজেক্টটি খুলে দেবে।

*   **টেস্ট করার জন্য:** অ্যান্ড্রয়েড স্টুডিওর রান বাটন চেপে আপনার ফোনে অথবা এমুলেটরে অ্যাপটি টেস্ট করতে পারেন।
*   **প্লে স্টোর ফাইলের জন্য (.aab jeni):** অ্যান্ড্রয়েড স্টুডিও থেকে **Build > Generate Signed Bundle / APK** এ যান। আপনার কি-স্টোর (KeyStore) ব্যবহার করে রিলিজ সাইনড **Bundle (.aab)** ফাইলটি বিল্ড করুন।
*   এই জেনারেটেড `.aab` ফাইলটি আপনি সহজেই **Google Play Console** এ আপলোড করে গুগল প্লে স্টোরে পাবলিশ করতে পারবেন।
